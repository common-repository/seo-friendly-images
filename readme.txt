== Description ==

SEO Friendly Images - New Version , Support Latest WP version 4.4. automatically adds alt and title attributes to all your images improving traffic from search engines.

1. Download plugin zip file to your computer and extract in a folder
1. Upload `seo-friendly-images` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
